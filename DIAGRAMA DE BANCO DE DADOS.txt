As relações entre as tabelas são as seguintes:

Users --  Personal_Data* - Um usuário pode responder a várias perguntas sobre a  Personal_Data, mas cada pergunta sobre a Personal_Data pertence a um único usuário.
Users -- Laranja*
Users -- Coco*
Users -- Limão*
Users -- Morango*
Users -- Maçã*

As chaves primárias e estrangeiras são as seguintes:

Users - id (chave primária)
Personal_Data - id (chave primária), id_user (chave estrangeira)
Laranja - id (chave primária), id_user (chave estrangeira)
Coco - id (chave primária), id_user (chave estrangeira)
Limão - id (chave primária), id_user (chave estrangeira)
Morango - id (chave primária), id_user (chave estrangeira)
Maçã - id (chave primária), id_user (chave estrangeira)

A requisicao do banco de dados ficaria: 

if ($allAnswered) {
    include_once('config.php');
    $atributoum = $_POST['atributoum'];
    $atributodois = $_POST['atributodois'];
    $atributotres = $_POST['atributotres'];
    $atributoquatro = $_POST['atributoquatro'];

    // Insere o registro na tabela "usuarios"
    $query = "INSERT INTO usuarios (atributoum) VALUES ('$atributoum')";
    $result = mysqli_query($conn, $query);

    // Obtém o ID do registro inserido
    $id_user = mysqli_insert_id($conn);

    // Insere o registro na tabela "uva"
    $query = "INSERT INTO Personal_Data (id_user, atributodois) VALUES ('$id_user', '$atributodois')";
    $result = mysqli_query($conn, $query);

    // Insere o registro na tabela "maca"
    $query = "INSERT INTO maca (id_user, atributotres) VALUES ('$id_user', '$atributotres')";
    $result = mysqli_query($conn, $query);

    // Insere o registro na tabela "coco"
    $query = "INSERT INTO coco (id_user, atributoquatro) VALUES ('$id_user', '$atributoquatro')";
    $result = mysqli_query($conn, $query);
}
