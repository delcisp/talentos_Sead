TESTES NECESSÁRIOS PARA APLICAÇÃO WEB, APLICAÇÃO QUE ENVOLVE BANCO DE DADOS ETC.

- TESTES DE UNIDADE/UNITÁRIOS (white-box) - Testar pedaços do código de forma isolada para garantir sua qualidade e funcionamento adequado.
- TESTES FUNCIONAIS (black-box) - Testar as funcionalidades do sistema do ponto de vista do usuário.
- TESTES DE INTEGRAÇÃO -  Verificar se diferentes partes ou unidades de código, que foram previamente testadas de forma isolada nos testes de unidade, funcionam corretamente juntas quando integradas em um sistema maior.

POSSÍVEIS FERRAMENTAS PARA USAR NA TESTAGEM: 

- JMeter
- Apache Benchmark
- Gatiling
(ferramentas tradicionais que oferecem controle, mas têm uma curva de aprendizado íngreme)

- LoaDView
- Load Impact
- LoadRunner Cloud
- BlazeMeter
(ferramentas sem código são mais fáceis de usar e adequadas para aqueles com experiência de programação limitada ou que precisam de configuração de teste rápido)
